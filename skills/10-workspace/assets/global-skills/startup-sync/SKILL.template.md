# 開工同步

在不改動專案或遠端狀態的前提下接續工作。

## 執行流程

1. 找出並讀取適用的 `AGENTS.md`，依序涵蓋工作區根目錄到目前目錄。
2. 讀取專案入口、固定路徑、安全規則與連結的 Obsidian 駕駛艙路徑。
3. 若已設定 Obsidian 駕駛艙，透過既有的 Obsidian 連線讀取。開工時不要建立或修改筆記。
4. 檢查 Git 分支、狀態與相關差異摘要。將所有既有變更視為使用者所有。
5. 回報目前狀態、未完成事項、風險，以及一項可立即執行的下一步。

## 安全規則

- 開工時不要拉取、提交、推送、部署或重寫檔案。
- 不要揭露檢查過程中發現的密鑰或學生姓名。
- 除非 chezmoi 已設定完成且使用者明確要求，否則不要檢查或執行 chezmoi。
- 若必要路徑或連線無法使用，回報問題並繼續完成其餘唯讀檢查。
